### Project Overview
This interactive Streamlit tool provides sports economic analysis for English Premier League clubs. Users can view revenue, attendance, wage bills, and profit margins across seasons and compare teams.

### Data Source
Kaggle Premier League Financial Dataset (Accessed: 19 April 2026)

### Features
- Key financial metrics display
- Historical revenue trend analysis
- Multi-team revenue comparison
- Interactive filters for team and year

### How to Run
1. Install required packages:
   pip install streamlit pandas matplotlib
2. Run the app:
   streamlit run app.py
