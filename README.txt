# Premier League Sports Economic Impact Analyzer

**Student Name**: Yongjin Chen

### Project Overview
This interactive Streamlit tool provides sports economic analysis for English Premier League clubs.
Users can view financial data (revenue, spending) and match performance (points, goals) across seasons.

### Data File
pl_economic_data.csv (2020-2023 season)

### How to Run (Teacher Guide)
1. Make sure ALL files are in the SAME folder
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run code_20260419.py

### Features
- Dataset overview
- Season & league filter
- Revenue vs Spending bar chart
- Points trend line chart
- Key findings & conclusions
